## Sobre o Projeto

O projeto analisa três frameworks que operam em camadas distintas da governança de TI:

- **COBIT 2019** — Governança e gestão: 5 domínios, 40 objetivos, cascata de metas, controles e indicadores (público: CIO, gestores, auditores).
- **ITIL 4** — Gerenciamento de serviços: SVS, cadeia de valor de serviço, 7 princípios orientadores e práticas operacionais (público: equipes de TI, service managers).
- **ISO/IEC 38500** — Governança corporativa de TI: 6 princípios, ciclo Avaliar → Dirigir → Monitorar (público: Conselho, Diretoria, C-Level).

**Tese central:** os três não se substituem — se complementam. A ISO 38500 define o *porquê* para o board. O COBIT operacionaliza o *o quê* para a gestão. O ITIL detalha o *como* para as equipes de serviço.
